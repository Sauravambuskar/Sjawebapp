SJA Foundation Platform Setup Guide

Follow these steps to set up your investment platform

This guide will help you set up the SJA Foundation Investment Management Platform.

Step 1: Create a Supabase Project

  1. Go to supabase.com and sign up or log in
  2. Create a new project
  3. Note down your project URL and anon key (found in Project Settings > API)

Step 2: Set Up the Database Schema

  1. In your Supabase project, go to the SQL Editor
  2. Open the supabase_schema.sql file from this project
  3. Copy the entire SQL script and paste it into the SQL Editor
  4. Run the script to create all required tables and functions
This step is critical! The platform will not work without the proper database schema.

Step 3: Create Storage Buckets

  1. In your Supabase project, go to Storage
  2. Create a new bucket named kyc_documents
  3. Create another bucket named payment_proofs
  4. Set both buckets to public (for this demo - in production, you should use more restrictive settings)

Step 4: Configure the Application

  1. Open assets/js/config.js in your code editor
  2. Update the Supabase URL and anon key with your own values
  3. Alternatively, when you first open the application, you'll be prompted to enter these values
// Default configuration
const defaultConfig = {
    supabaseUrl: 'YOUR_SUPABASE_URL',
    supabaseAnonKey: 'YOUR_SUPABASE_ANON_KEY'
};

Step 5: Run the Application

  1. Install dependencies: npm install
  2. Start the application: npm start
  3. Open your browser and navigate to http://localhost:3000

Step 6: Create an Admin User

  1. Register a new user through the application
  2. In your Supabase project, go to the Table Editor
  3. Open the users table
  4. Find your user and change the role field from client to admin
  5. Log out and log back in to access the admin dashboard
